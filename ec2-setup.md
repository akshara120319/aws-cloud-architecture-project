# EC2 Instance Setup

## Objective
To create an EC2 instance for hosting the application.

## Steps Followed
1. Logged into AWS Management Console.
2. Opened EC2 service.
3. Clicked on Launch Instance.
4. Selected Amazon Linux 2 AMI.
5. Chose t2.micro instance type.
6. Configured security group to allow HTTP and SSH.
7. Launched the instance using a key pair.

## Outcome
EC2 instance was successfully launched and running.
