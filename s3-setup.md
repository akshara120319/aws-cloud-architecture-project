# Amazon S3 Bucket Setup

## Objective
To create an S3 bucket for storing static files and backups.

## Steps Followed
1. Logged into AWS Management Console.
2. Opened Amazon S3 service.
3. Clicked on Create bucket.
4. Entered a unique bucket name.
5. Selected the same region as other AWS resources.
6. Kept Block Public Access enabled for security.
7. Created the bucket.

## Outcome
S3 bucket was successfully created and is ready to store files.
