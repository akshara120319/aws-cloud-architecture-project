# Amazon RDS Database Setup

## Objective
To create a managed relational database for storing application data.

## Steps Followed
1. Opened Amazon RDS service from AWS Console.
2. Clicked on Create database.
3. Selected MySQL as the database engine.
4. Chose Free Tier template.
5. Provided DB identifier, username, and password.
6. Selected default VPC for connectivity.
7. Disabled public access for security.
8. Created the database instance.

## Outcome
RDS database was successfully created and became available after setup.
