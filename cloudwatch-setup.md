# CloudWatch Monitoring Setup

## Objective
To monitor AWS resources and track system performance.

## Steps Followed
1. Opened Amazon CloudWatch from AWS Console.
2. Navigated to Metrics section.
3. Selected EC2 metrics.
4. Viewed CPU Utilization for the EC2 instance.
5. Created alarms for high CPU usage (optional).
6. Checked logs for EC2 and Auto Scaling activities.

## Outcome
CloudWatch monitoring was successfully enabled and system metrics
are being tracked.
